# Employee Master Data Management (MDM) using Snowflake and SQL

## Overview

This project demonstrates an end-to-end Employee Master Data Management (MDM) solution implemented using Snowflake and SQL. The objective is to integrate employee data from multiple source systems, eliminate duplicate records, improve data quality, and create a single trusted Golden Record for each employee.

This project was completed as part of my Data Engineering Internship at Systech Solutions Pvt. Ltd.

---

## Problem Statement

Employee information is collected from multiple source systems such as HRMS, Payroll, Active Directory, and CRM. Due to inconsistencies in data formats and duplicate records, maintaining accurate employee information becomes challenging.

The objective of this project is to consolidate employee data, perform data cleansing and standardization, apply matching and survivorship rules, and generate a trusted Golden Record for every employee.

## Project Workflow

1. Load employee datasets into Snowflake.
2. Create source and staging tables.
3. Integrate data from multiple source systems.
4. Perform data cleansing and standardization.
5. Apply matching and survivorship rules.
6. Generate Golden Records.
7. Validate results using a Data Quality Report.
